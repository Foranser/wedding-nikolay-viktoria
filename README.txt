Николай и Виктория — 08.08.2026

Файлы:
1. index.html — основная страница сайта.
2. style.css — оформление сайта.
3. script.js — работа формы обратной связи.
4. google-apps-script.js — код для Google Apps Script, чтобы ответы попадали в Google Таблицу.

КАК ПОДКЛЮЧИТЬ СБОР ОТВЕТОВ В GOOGLE ТАБЛИЦУ:

1. Создайте новую Google Таблицу.
2. В первой строке добавьте заголовки:
   Дата ответа | Имя | Присутствие | Части мероприятия | Дети | Питание | Комментарий
3. В Google Таблице откройте: Расширения → Apps Script.
4. Удалите стандартный код и вставьте код из файла google-apps-script.js.
5. Нажмите Развернуть → Новое развертывание.
6. Тип приложения: Веб-приложение.
7. Доступ: Все пользователи.
8. Скопируйте ссылку веб-приложения.
9. Откройте файл script.js и вставьте ссылку в строку:
   const GOOGLE_SCRIPT_URL = '';

После этого ответы гостей будут отправляться в Google Таблицу.

Обновление v13: исправлена отправка формы в Google Таблицу.
Теперь данные отправляются через URLSearchParams, а Apps Script читает e.parameter.
Это стабильнее для Google Apps Script и помогает избежать проблем с CORS/JSON.

После замены google-apps-script.js в Apps Script обязательно:
1. нажмите «Сохранить»;
2. нажмите «Развернуть»;
3. выберите «Управление развертываниями»;
4. нажмите карандаш;
5. выберите «Новая версия»;
6. нажмите «Развернуть»;
7. скопируйте актуальную ссылку веб-приложения и вставьте её в script.js.


Обновление v14: исправлена проблема, когда строки в Google Таблице появлялись, но поля были пустые.

При обычном открытии ссылки Apps Script пустые строки больше не создаются.
